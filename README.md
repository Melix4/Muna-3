Проект команды "Муна-3" 
Предмет: Введение в авиационную и ракетно-космическую технику

Программная реализация моделирования полета. 
Файл constants.py содержит физические константы
grapics.py - запустить для построения графиков
main.py - программа, проводящая основные рассчеты. Записывает данные в excel-файл

Для установки необходимых библиотек нужно выполнить команду
pip install -r requirements.txt
